# Sample input format

```
【目标读者】
程序员 / AI工程师 / OpenClaw用户 / 技术管理者

【标题】
AI 没有替代程序员，只是把程序员重新分层了

【封面】
/home/ubuntu/.hermes/research/imgs/cover.png
（或外链URL: http://mmbiz.qpic.cn/.../0?wx_fmt=jpeg）

【已知参考数据】
阅读 1234 / 点赞 56 / 分享 12 / 推荐 23
（首次发布可空着）
```

## 期望输出结构

skill 会按以下结构返回（详见 SKILL.md）：

1. **总分 X/100**
2. **分项打分**（七维，每维+1句扣分理由）
3. **流量判断**（高/中高/中等/偏弱/很弱 + 一句话）
4. **最大问题**（1-2条）
5. **改法建议**：
   - 3个新标题（必须可直接用）
   - 1个封面优化方向
   - 1句封面主文案（≤15字）
6. **特别检查**（4条红旗按需勾选）

## 典型调用场景

| 场景 | 调用方式 |
|---|---|
| 写完文章准备发 | "帮我给这篇打分：[目标读者] / [标题] / [封面路径]" |
| 多标题比稿 | "我有3个备选标题，请按'AI 重新分层程序员'这篇打分，给出最高分的那个" |
| 复盘爆款 | "这篇阅读5w，帮我分析为什么火：[链接]" |
| 复盘扑街 | "这篇阅读200，帮我看哪里有问题：[标题] / [封面]" |
